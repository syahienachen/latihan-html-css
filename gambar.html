Gambar

Pada HTML untuk menampilkan sebuah gambar kita bisa menggunakan tag <img>. Berbeda dengan elemen lain, elemen &kt;img> tidak menuliskan konten diantara tag pembuka dan tag penutup sebagaimana tag lainnya. Namun untuk menetapkan gambar yang ditampilkan kita gunakan sebuah atribut src. Contohnya seperti ini:

<img src="https://cdn.logo.com/hotlink-ok/logo-social.png">

Hasil:

Pada elemen <img> terdapat 5 atribut yaitu:

  1. src
  2. alt
  3. title
  4. width
  5. height

Atribut src

atribut src. Atribut ini berfungsi sebagai sumber dari gambar yang ditampilkan. Atribut ini dapat bernilai url gambar atau path gambar lokal dari gambar yang digunakan.

Atribut alt

Atribut ini sebenarnya tidak wajib untuk diterapkan, hanya saja atribut ini akan sangat berguna ketika gambar tidak berhasil ditampilkan. Nilai atribut ini merupakan gambaran dari gambar yang ditampilkan dalam bentuk tulisan. Sehingga ketika gambar gagal ditampilkan maka akan memunculkan teks alternatif yang dapat mewakili arti dari gambar tersebut.

Atribut title

Atribut title berfungsi sebagai informasi tambahan untuk sebuah gambar. Informasi tersebut akan muncul ketika kita mengarahkan sebuah cursor pada gambar yang ditampilkan.

Atribut width

Untuk menentukan lebar gambar kita gunakan atribut width

Atribut height

Untuk menentukan tinggi gambar kita gunakan atribut height

Jenis Format Gambar

  1. Graphics Interchange Format (.gif)
  2. Joint Photographic Expert Group image (.jpg, .jpeg, .jfif, .pjpeg, .pjp)
  3. Portable Network Graphics (.png)
  4. WebP (.webp)
  5. Scalable Vector Graphics (.svg)

Mengatur Ukuran Gambar

Ketika menggunakan atribut width dan height disarankan hanya gunakan salah satunya. Terkecuali kita menentukan nilai lebar dan tingginya sesuai dengan rasio dari ukuran gambar aslinya.

Contohnya, jika kita tetap memaksa untuk menentukan ukuran panjang dan lebar sebuah gambar tanpa menyesuaikan rasionya, maka gambar yang ditampilkan tidak akan proporsional.

<img src="https://cdn.logo.com/hotlink-ok/logo-social.png" alt="gambar logo" title="gambar logo" width="300px" height="50px"><!--Jangan Lakukan Seperti ini-->

Menetapkan ukuran gambar berdasarkan lebarnya

Dengan menetapkan hanya satu atribut ukurannya, maka ukuran lainnya akan mengikuti sesuai dengan rasio gambar aslinya. Contohnya, kita bisa menetapkan ukuran gambar berdasarkan nilai lebarnya saja.
